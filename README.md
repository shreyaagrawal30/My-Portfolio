# Shreya Agrawal | Personal Portfolio 🚀

A modern, responsive portfolio website built with **React**, **Vite**, and **Framer Motion**. This project showcases my work as a Data Analyst and Software Developer, featuring smooth scrolling, a dynamic project grid, and a functional contact form.

## 🔗 Live Demo
Check out the live site here: [shreyaagrawal.vercel.app](https://shreyaagrawal.vercel.app/)

---

## 🛠️ Tech Stack
* **Frontend:** React.js, Vite
* **Animations:** Framer Motion
* **Styling:** Pure CSS3 (Custom Properties / Media Queries)
* **Icons:** DevIcon, FontAwesome
* **Deployment:** Vercel

---

## 🚀 Getting Started

To run this project locally on your machine, follow these steps:

### 1. Clone the repository
```
git clone https://github.com/shreyaagrawal30/Shreya_Portfolio.git
cd Shreya_Portfolio/portfolio
```
### 2. Install Dependencies 
```
npm install
```
### 3. Run the Development Server
```
npm run dev
```
### 4. Build for Production
```
npm run build
```

### 📁 Project Structure
* /src
  * /assets -> Images
  * /components -> React components (Hero, About, Projects, etc.)

* /public -> Static assets and downloadable resume

* App.jsx -> Main application layout
